Музыкальный бот для Discord.

1. О программе:

Данная программа была создана для голосовых чатов Discord. 
Вы можете подключить с помощью команд к голосовому каналу бота, и он будет воспроизводить песню/музыку/подкаст по ссылке с Youtube. Также доступны и другие команды, такие как: пауза, продолжение воспроизведения, отключение бота от канала и тд.

2. Запуск программы:

- Создайте бота в дискорде, скопируйте его токен.
- Перейдите по ссылке Git-hub, скачайте оттуда все файлы и переместите в проект Pycharm: https://github.com/AnnaMasia/MusiyachenkoDiscordbotproject
- Откройте файл 'requirements.txt' и скачайте все предложенные библиотеки.
- Вставьте токей вашего бота в файл "config.py"
- Запустите скаченный файл 'main.py'.
- Готово!
